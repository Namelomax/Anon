# Пользовательский словарь терминов, которые ВСЕГДА маскируются, даже если
# NER/LLM их не распознают: аббревиатуры, внутренние прозвища организаций,
# обычные слова, которые в контексте конкретного заказчика называют
# конкретное ведомство/организацию.
#
# Формат (одна запись на строку):
#   Алиас1, Алиас2, ... = Каноничное имя
# - Алиасы через запятую — любой из них будет замаскирован.
# - Каноничное имя — то, что попадёт в mapping.json как "оригинал"
#   (не обязано совпадать ни с одним алиасом дословно).
# - Регистр не важен, падежные окончания (Минфина, Минфином...) подхватываются
#   автоматически — писать варианты падежей не нужно.
# - Строки, начинающиеся с #, и пустые строки игнорируются.

Мингос = Министерство государственного управления
Минфин = Министерство финансов
Правительство = Правительство
